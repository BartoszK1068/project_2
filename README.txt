Projekt U3 - prosta wersja

Pliki:
- metoda.py - uniwersalna implementacja drzewa decyzyjnego
- problem.py - dane o znakach drogowych i warianty atrybutow
- sterowanie.py - menu tekstowe, uruchamianie i zapis do logu
- main.py - punkt wejscia
- dane/road_signs.csv - przykladowy zbior danych

Uruchomienie:
python main.py

Program zapisuje:
- log wykonania do pliku monitor.log
- zbudowane drzewa do folderu wyniki_drzew

Dostepne warianty:
- full
- without_crossed_out
- without_element_count
- without_shape_and_border
